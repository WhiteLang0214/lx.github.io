VUE实战第三天：高仿饿了么APP---Vue-cli开始项目

2017-07-24

vue-cli脚手架工具

- 帮助我们写好vue.js基础代码的工具,帮助我们搞定
- 目录结构
- 本地调试
- 代码部署
- 热加载
- 单元测试

安装vue-cli

- 保障node.js在版本4.4.5以上
-

安装

- $npm install -g vue-cli

初始化webpack打包项目名

- $vue init webpack 项目名

切换到目录

- $cd 项目名

安装依赖

- $npm install
- 安装成功后,项目中增加文件:node_modules

运行

- $npm run dev
- 如果出现listening at http: // localhost: 4000就说明启动成功了

项目文件

一级目录

1. build

2. config

3. node_modules

- 通过npm install 安装的依赖代码库

4. src

- 存放的源码

5. static

- 存放的静态资源

.babelIrc文件

  1. babel-plugin-transform-es2015-block-scopped-function文件将es6的语法转换为es5的语法
  2. babel-plugin-transform-es2015-classes文件将class编译成es5编译类的方式
  3. babel-plugin-transform-runtime—>lib—>definitions.js文件对es6的symble

.editorconfig文件

  1. 编辑器一些的配置

.eslintignore文件

  1. 忽略语法检查的目录文件
  2. 主要对build和config文件中的js做es语法检查

    .eslintsrc.js

  3. extends: ‘standard’ 使用标准的语法标准
  4. rules可以修改规则
    为:0的就是忽略语法检查
    1. ‘arrow-parens’:对箭头函数的规则

index.html

  1. 入口html文件
  2. webpack编译时,会将项目自动插入到index.html中

main.js

  1. 入口文件的js
