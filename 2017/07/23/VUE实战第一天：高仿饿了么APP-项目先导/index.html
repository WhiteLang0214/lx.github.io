VUE实战第一天：高仿饿了么APP---项目先导

2017-07-23

学习目标

  • 掌握vue.js在实战中的应用
  • 学会使用vue.js完整地开发移动端APP
  • 学会组件化,模块化的开发方式

学习内容

  1. vue.js框架介绍
  2. vue-cli脚手架 搭建基本代码框架
  3. vue-router 官方插件管理路由
  4. vue-resource Ajax通信
  5. webpack 构建工具
  6. es6 + eslint eslint : es6代码风格检查工具
  7. 工程化 组件化 模块化
  8. 移动端常用的开发技巧
    • flex弹性布局
    • css stickyfooter
    • 酷炫的交互设计

流程

  1. 需求分析
  2. 脚手架工具
  3. 数据mock
  4. 架构设计
  5. 代码编写
  6. 自测
  7. 编译打包

要求

  1. 代码开发及测试环节
    • UI标注
    • 真是数据演示
  2. 代码规范
    • 架构设计
    • 组件抽象
    • 模块拆分
    • 代码风格统一
    • JS变量命名规范
    • CSS代码规范

功能技术分析

  1. 后端—>用vue-resource做数据交互
  2. 前端—>用vue-router实现单页应用
  3. 用第三方js库better-scroll实现最大程度组件化,重复的CSS可以最大程度复用
  4. 收藏商家功能—>用html5的localstorge
  5. 图标字体的使用
  6. 移动端1像素边框
  7. css sticky footer布局
  8. flex弹性布局
